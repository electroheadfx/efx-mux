**Gap 14 -- Tauri project status indicator (UAT test 14, MAJOR):**

Root cause: The Rust waiter thread uses `libc::waitpid(pid)` to detect process exit. For multi-stage commands like `pnpm tauri dev`, the shell (`sh -c`) spawns pnpm, which spawns node, which spawns the Vite dev server, then pnpm spawns `cargo tauri dev`, which spawns the Tauri app. The initial `sh` process or intermediate processes may exit before the actual server is ready, causing `waitpid` to fire prematurely with a non-zero or zero exit code. This triggers `server-stopped` on the frontend, which flips status to 'crashed' (showing red) even though the server is still starting up.

**Rust fix in `src-tauri/src/server.rs`:**

Replace the waitpid-based waiter thread with a stdout/stderr EOF-based approach. The key insight: stdout and stderr pipes stay open as long as ANY process in the process group has them open (inherited file descriptors). When ALL processes exit, the pipes close and the reader threads get EOF.

1. Remove the current waitpid waiter thread entirely (lines 86-102).

2. Instead, use the stdout and stderr reader threads to detect process exit. Modify the approach:
   - Create an `Arc<AtomicU8>` counter initialized to the number of active readers (2 if both stdout and stderr, 1 if only one).
   - Each reader thread, after its EOF loop exits, decrements the counter.
   - When the counter reaches 0 (last reader to finish), that thread emits `server-stopped` with the exit code.
   - To get the exit code after all pipes close, call `child.wait()` (or `waitpid` at that point -- the process is definitely done).

Here is the concrete implementation. Replace the stdout/stderr spawn + waitpid section with:

```rust
use std::sync::Arc;
use std::sync::atomic::{AtomicU8, Ordering};

// After taking stdout/stderr and storing child:
let reader_count = Arc::new(AtomicU8::new(
    (stdout.is_some() as u8) + (stderr.is_some() as u8)
));

// If no readers at all (shouldn't happen but guard), emit stopped immediately
if reader_count.load(Ordering::SeqCst) == 0 {
    let _ = app.emit("server-stopped", 0i32);
    return Ok(());
}

// Spawn stdout reader thread
if let Some(mut stdout) = stdout {
    let app_clone = app.clone();
    let count = reader_count.clone();
    let pid_for_wait = pid;
    std::thread::spawn(move || {
        let mut buf = [0u8; 4096];
        loop {
            match stdout.read(&mut buf) {
                Ok(0) => break,
                Ok(n) => {
                    let text = String::from_utf8_lossy(&buf[..n]).to_string();
                    let _ = app_clone.emit("server-output", &text);
                }
                Err(_) => break,
            }
        }
        // Last reader to finish emits server-stopped
        if count.fetch_sub(1, Ordering::SeqCst) == 1 {
            // All pipes closed -- get exit status
            let mut status: libc::c_int = 0;
            let result = unsafe { libc::waitpid(pid_for_wait as i32, &mut status, libc::WNOHANG) };
            let exit_code = if result > 0 && libc::WIFEXITED(status) {
                libc::WEXITSTATUS(status)
            } else {
                0 // Pipes closed, process exited normally
            };
            let _ = app_clone.emit("server-stopped", exit_code);
            // Clear stored child
            if let Ok(mut guard) = app_clone.state::<ServerProcess>().0.lock() {
                *guard = None;
            }
        }
    });
}

// Spawn stderr reader thread (same pattern)
if let Some(mut stderr) = stderr {
    let app_clone = app.clone();
    let count = reader_count.clone();
    let pid_for_wait = pid;
    std::thread::spawn(move || {
        let mut buf = [0u8; 4096];
        loop {
            match stderr.read(&mut buf) {
                Ok(0) => break,
                Ok(n) => {
                    let text = String::from_utf8_lossy(&buf[..n]).to_string();
                    let _ = app_clone.emit("server-output", &text);
                }
                Err(_) => break,
            }
        }
        if count.fetch_sub(1, Ordering::SeqCst) == 1 {
            let mut status: libc::c_int = 0;
            let result = unsafe { libc::waitpid(pid_for_wait as i32, &mut status, libc::WNOHANG) };
            let exit_code = if result > 0 && libc::WIFEXITED(status) {
                libc::WEXITSTATUS(status)
            } else {
                0
            };
            let _ = app_clone.emit("server-stopped", exit_code);
            if let Ok(mut guard) = app_clone.state::<ServerProcess>().0.lock() {
                *guard = None;
            }
        }
    });
}
```

Add the necessary imports at the top of `server.rs`: `use std::sync::atomic::{AtomicU8, Ordering};` (Arc is already imported via `std::sync::Mutex`... check -- if `Arc` is not imported, add `use std::sync::Arc;`).

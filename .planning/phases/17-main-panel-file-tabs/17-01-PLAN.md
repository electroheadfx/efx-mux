**Step 1: Create `src/editor/setup.ts`** -- CM6 EditorState factory.

Export `function createEditorState(content: string, options: EditorSetupOptions): EditorState` where:

```typescript
interface EditorSetupOptions {
  language: Extension | null;
  onSave: (content: string) => void;
  onDirtyChange: (dirty: boolean) => void;
}
```

The function creates `EditorState.create({ doc: content, extensions: [...] })` with these extensions (per D-06):
1. `basicSetup` from `codemirror` (includes line numbers, active line, bracket matching, auto-close, search Cmd+F, code folding)
2. The `language` extension from options (if not null)
3. `efxmuxTheme` from `./theme`
4. `syntaxHighlighting(efxmuxHighlightStyle)` from `@codemirror/language` wrapping the highlight style from `./theme`
5. `keymap.of([{ key: 'Mod-s', run: (view) => { options.onSave(view.state.doc.toString()); return true; } }])` from `@codemirror/view`
6. `EditorView.updateListener.of((update) => { if (update.docChanged) { options.onDirtyChange(update.state.doc.toString() !== savedContent); } })` -- the `savedContent` is captured as a closure over `content` parameter
7. `showMinimap.compute(['doc'], () => ({ create: () => ({ dom: document.createElement('div') }), displayText: 'blocks', showOverlay: 'always' }))` from `@replit/codemirror-minimap`

Also export `function updateSavedContent(state: EditorState, newContent: string): void` -- This is a noop placeholder; dirty tracking uses a module-level WeakMap keyed by EditorState to track saved content per editor instance. Actually, simpler approach: export a `savedContentMap: WeakMap<EditorView, string>` that EditorTab can use to update after save.

Correction -- keep it simpler: do NOT use WeakMap. The `createEditorState` function returns `{ state: EditorState, getSavedContent: () => string, setSavedContent: (s: string) => void }` using a closure:

```typescript
export interface EditorSetupResult {
  state: EditorState;
  getSavedContent: () => string;
  setSavedContent: (content: string) => void;
}

export function createEditorState(content: string, options: EditorSetupOptions): EditorSetupResult {
  let savedContent = content;
  const state = EditorState.create({
    doc: content,
    extensions: [
      basicSetup,
      ...(options.language ? [options.language] : []),
      efxmuxTheme,
      syntaxHighlighting(efxmuxHighlightStyle),
      keymap.of([{
        key: 'Mod-s',
        run: (view) => { options.onSave(view.state.doc.toString()); return true; },
      }]),
      EditorView.updateListener.of((update) => {
        if (update.docChanged) {
          options.onDirtyChange(update.state.doc.toString() !== savedContent);
        }
      }),
      showMinimap.compute(['doc'], () => ({
        create: () => ({ dom: document.createElement('div') }),
        displayText: 'blocks',
        showOverlay: 'always',
      })),
    ],
  });
  return {
    state,
    getSavedContent: () => savedContent,
    setSavedContent: (s: string) => { savedContent = s; },
  };
}
```

Import `basicSetup` from `codemirror`. Import `EditorState` from `@codemirror/state`. Import `EditorView, keymap` from `@codemirror/view`. Import `syntaxHighlighting` from `@codemirror/language`. Import `showMinimap` from `@replit/codemirror-minimap`.

```tsx

// sub: text-[10px], textDim, fontFamily: mono

//   dot: 6px circle, statusGreen or diffRed
//   text: text-[10px] mono font-medium, statusGreen or diffRed
//   label: isRunning ? 'Ready' : 'Stopped'
```

## Status pill pattern (from agent-header.tsx reference)
The status pill uses `bg-success/[0.125]` and `bg-danger/[0.125]` Tailwind arbitrary values. In the new palette these must use `tokens.ts` inline styles:
- Running: `style={{ backgroundColor: colors.statusGreenBg, borderRadius: 4, padding: '3px 8px', gap: 4 }}`
- Stopped: `style={{ backgroundColor: colors.diffRedBg, borderRadius: 4, padding: '3px 8px', gap: 4 }}`

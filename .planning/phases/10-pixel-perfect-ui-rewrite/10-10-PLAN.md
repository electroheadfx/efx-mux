# Design principle for light mode
The dark mode uses navy-blue as the dominant background. The light mode should use a neutral gray-white palette that is clearly different from the dark mode but harmonizes with the accent color (#258AD1).

Good light mode values for developer tools:
- Background: #FFFFFF (pure white) or #F6F8FA (light gray)
- Raised surfaces: #FFFFFF or #EBEEF3
- Border: #D0D7DE (GitHub light border is fine)
- Text: #1F2328 (near black)
- Accent: keep #258AD1 (same blue works in light mode)
- Success: #1A7F37
- Warning: #9A6700
- Danger: #CF222E

<objective>
Fix extra newlines appearing in terminal when app opens with Claude Code agent.

The issue is caused by the `clear` command in the shell wrapper at line 93 of pty.rs. The wrapper runs:
```
/bin/zsh -c 'clear; {agent}; exec /bin/zsh'
```

The `clear` command can produce extra newlines depending on terminal state/PT Y dimensions at initialization time.
</objective>

<context>
@src-tauri/src/terminal/pty.rs
</context>

<tasks>

<task type="auto">
  <name>Remove clear command from shell wrapper</name>
  <files>src-tauri/src/terminal/pty.rs</files>
  <action>
    In the spawn_terminal function, modify the shell_command wrapper at line 93.
    Change from:
    ```
    let wrapped = format!("{} -c 'clear; {}; exec {}'", user_shell, shell_cmd, user_shell);
    ```
    To:
    ```
    let wrapped = format!("{} -c '{}; exec {}'", user_shell, shell_cmd, user_shell);
    ```
    This removes the `clear;` prefix that was causing extra newlines on startup.
  </action>
  <verify>
    <automated>grep -n "clear;" src-tauri/src/terminal/pty.rs || echo "clear removed successfully"</automated>

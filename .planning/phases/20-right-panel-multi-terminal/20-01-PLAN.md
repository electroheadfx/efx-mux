<objective>
Refactor `src/components/terminal-tabs.tsx` from a single-instance module (main only) into a scope-parametrized module keyed on `TerminalScope = 'main' | 'right'`, while preserving every existing top-level export as a thin wrapper resolving to scope `'main'` (D-11 backward compatibility). Expose a new `getTerminalScope(scope)` accessor for right-panel call sites. Fix the crash-restart suffix collision (Pitfall 1) by renaming restart suffix from `r<N>` to `rr<N>` universally.

Purpose: Enables Plan 04's right-panel rewrite to spawn independent terminal tabs in the right pane without any code duplication, while keeping Phase 17's main-panel code path completely untouched at call sites.

Output: A refactored `terminal-tabs.tsx` with a `scopes: Map<TerminalScope, ScopeState>` registry, scoped internal functions (`createNewTabScoped`, `closeTabScoped`, `switchToTabScoped`, etc.), scoped persistence keys (`terminal-tabs:<project>` for main, `right-terminal-tabs:<project>` for right), and a new `terminal-tabs.test.ts` validating scope isolation, session naming, crash-restart suffix, and persistence round-trip.
</objective>

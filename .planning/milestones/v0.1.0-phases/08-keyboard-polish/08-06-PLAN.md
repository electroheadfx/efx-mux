**Bug 1 - Agent resolution (line 129):**
`createNewTab()` calls `resolveAgentBinary(projectInfo?.agent)` for ALL tabs. Only the first tab (created via `initFirstTab`) should get the agent binary. Subsequent tabs from Ctrl+T should always be plain shell.

Fix: Remove the `resolveAgentBinary` call from `createNewTab()`. Change line 129 to:
```typescript
const agentBinary = undefined; // Ctrl+T tabs are always plain shell (UAT gap 1)
```
Also remove the `import { detectAgent }` if it's only used here (check -- it's also used in `restartTabSession`, so keep the import but don't call it in createNewTab).

**Bug 2 - switchToTab reflow (lines 320-332):**
After setting `display: block`, `tab.terminal.focus()` and `tab.fitAddon.fit()` fire synchronously before the browser has reflowed. `fit()` reads zero dimensions, terminal appears invisible.

Fix: Wrap the focus+fit in `requestAnimationFrame`:
```typescript
function switchToTab(tabId: string): void {
  const tabs = terminalTabs.value;
  for (const tab of tabs) {
    if (tab.id === tabId) {
      tab.container.style.display = 'block';
    } else {
      tab.container.style.display = 'none';
    }
  }
  // Defer focus+fit until after browser reflow (UAT gap 3)
  requestAnimationFrame(() => {
    const active = tabs.find(t => t.id === tabId);
    if (active) {
      active.fitAddon.fit();
      active.terminal.focus();
    }
  });
}
```
Note: fit() before focus() -- fit needs correct dimensions before focus triggers cursor rendering.

**In terminal-tabs.tsx -- add restoreTabs() function:**

Export a new `restoreTabs` function that:
1. Accepts saved tab data (parsed from state.json `session['terminal-tabs']`).
2. For each saved tab:
   a. Creates a container div, appends to `.terminal-containers`.
   b. Creates terminal + fitAddon via `createTerminal()`.
   c. Registers with theme manager.
   d. Connects PTY via `connectPty(terminal, savedTab.sessionName, projectPath, undefined)` -- NO agent binary for restored tabs (agent is only on first init).
   e. For the FIRST restored tab only: if the project has an agent configured, resolve and pass the agent binary (since the first tab is the agent tab).
   f. Attaches resize handler.
   g. Builds TerminalTab object with the saved sessionName and label.
3. Sets `terminalTabs.value` to the restored array.
4. Sets `activeTabId.value` to saved activeTabId (or first tab if saved ID not found).
5. Calls `switchToTab` on the active tab.
6. Updates `tabCounter` to the count of restored tabs.

Signature:
```typescript
export async function restoreTabs(
  savedData: { tabs: Array<{ sessionName: string; label: string }>; activeTabId: string },
  projectPath?: string,
  agentBinary?: string,
): Promise<boolean>
```
Returns true if restoration succeeded (at least 1 tab restored), false otherwise.

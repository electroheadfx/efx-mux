# Requirements: Efxmux v0.3.0 (ARCHIVED)

**Archived:** 2026-04-20
**Status:** SHIPPED

## v0.3.0 Requirements

All requirements for Workspace Evolution release completed.

### File Editing (EDIT)

- [x] **EDIT-01**: User can open files in main panel tabs with CodeMirror 6 syntax highlighting
- [x] **EDIT-02**: User sees unsaved indicator (dot) in tab title when file has uncommitted changes
- [x] **EDIT-03**: User can save file with Cmd+S keyboard shortcut
- [x] **EDIT-04**: User sees confirmation modal when closing tab with unsaved changes
- [x] **EDIT-05**: User can reorder tabs via drag and drop

### Git Control (GIT)

- [x] **GIT-01**: User can stage individual files via checkboxes in git control pane
- [x] **GIT-02**: User can unstage individual files via checkboxes
- [x] **GIT-03**: User can commit staged changes with message input
- [x] **GIT-04**: User can push commits to remote repository
- [x] **GIT-05**: User can undo last commit (soft reset)

### File Tree (TREE)

- [x] **TREE-01**: User can delete files/folders via context menu with confirmation dialog
- [x] **TREE-02**: User can delete files/folders via Delete key with confirmation dialog
- [x] **TREE-03**: User can open file in external editor (Zed, VSCode) via context menu
- [x] **TREE-04**: User can drag/drop files and folders to reorder within tree
- [x] **TREE-05**: User can drag files from Finder into tree to import

### GSD Viewer (GSD)

- [x] **GSD-01**: User can view Milestones sub-tab parsed from ROADMAP.md
- [x] **GSD-02**: User can view Phases sub-tab parsed from ROADMAP.md
- [x] **GSD-03**: User can view Progress sub-tab parsed from ROADMAP.md
- [x] **GSD-04**: User can view History sub-tab from MILESTONES.md
- [x] **GSD-05**: User can view State sub-tab (current position + decisions) from STATE.md

### Sidebar (SIDE)

- [x] **SIDE-01**: User can switch between 3 sidebar tabs: Projects, File Tree, Git Control
- [x] **SIDE-02**: User can add Terminal/Agent sub-TUI via plus menu in right panel

### Main Panel (MAIN)

- [x] **MAIN-01**: User can add new tabs via dropdown menu (Terminal Zsh, Agent, Git changes)
- [x] **MAIN-02**: User can view git changes panel with accordion per-file diffs
- [x] **MAIN-03**: User can create new file from folder context in file tree

### Bug Fixes (FIX)

- [x] **FIX-01**: File tree updates when files are changed by external editors
- [~] **FIX-02**: ~~No phantom characters during fast terminal scroll~~ -- SUPERSEDED (not reproducible)
- [~] **FIX-03**: ~~Terminal has thin draggable scrollbar~~ -- SUPERSEDED (acceptable as-is)
- [~] **FIX-04**: ~~Sidebar bottom TUI padding/alignment~~ -- SUPERSEDED (bottom TUI removed)
- [x] **FIX-05**: Open-in-external-editor launches chosen editor reliably
- [x] **FIX-06**: Clicking files in file tree opens them in editor tabs

### Dynamic Workspace (Phase 22)

- [x] **TABS-01**: Dynamic tabs (GSD, Git Changes) are closeable, reorderable, addable via + button
- [x] **SPLIT-01**: Main and right panels support 1-3 stacked sub-panes
- [x] **SPLIT-02**: Intra-zone resize handles work between sub-panes
- [x] **SPLIT-03**: Tabs can be dragged between any two scopes
- [x] **SPLIT-04**: PTY session names remain stable across drags
- [x] **PREF-01**: Preferences button in titlebar opens preferences modal

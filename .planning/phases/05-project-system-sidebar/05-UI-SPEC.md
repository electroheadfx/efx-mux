# Phase 5 — UI Design Contract

> Visual and interaction contract for the project system sidebar, project registration modal, fuzzy search overlay, and git status integration. Generated by gsd-ui-researcher, verified by gsd-ui-checker.

---

## Design System

| Property | Value |
|----------|-------|
| Tool | none (Arrow.js + plain CSS custom properties) |
| Preset | not applicable |
| Component library | none (Arrow.js `html` template literals) |
| Icon library | Unicode symbols (existing pattern from Phase 1 sidebar) |
| Font | FiraCode Light 300, 14px base (loaded via @font-face in theme.css) |

---

## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding, terminal-area padding |
| sm | 8px | Sidebar content padding, nav item gaps, collapsed icon gaps |
| md | 16px | Modal field spacing, section padding |
| lg | 24px | Modal internal padding, section breaks |
| xl | 32px | Modal outer margin from viewport edge |
| 2xl | 48px | Not used this phase |
| 3xl | 64px | Not used this phase |

Exceptions: Sidebar nav items use 6px vertical padding (existing pattern from Phase 1, keep for density).

---

## Typography

| Role | Size | Weight | Line Height | Usage |
|------|------|--------|-------------|-------|
| Body | 14px | 300 (Light) | 1.5 | Project names in expanded sidebar, modal field labels |
| Label | 11px | 300 (Light) | 1.4 | Section headers (PROJECTS, GIT CHANGES), badge text, git branch name |
| Small | 12px | 300 (Light) | 1.4 | File change counts, toast message, fuzzy search result secondary text |
| Heading | 16px | 300 (Light) | 1.2 | Modal title ("Add Project"), fuzzy search input text |

Note: All text uses `letter-spacing: 0.08em` and `text-transform: uppercase` for section headers only (existing EFXMUX header pattern).

---

## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `#282d3a` (var(--bg)) | App background, sidebar background, modal backdrop base |
| Secondary (30%) | `#363b3d` (var(--bg-raised)) | Modal surface, toast surface, fuzzy search overlay surface, active project row hover |
| Accent (10%) | `#258ad1` (var(--accent)) | Active project left border indicator, Ctrl+P selected result highlight, primary CTA button background, git branch badge text |
| Destructive | `#dc322f` (Solarized Red) | Remove project button, error toast left border |
| Text primary | `#92a0a0` (var(--text-bright)) | Project names, modal title, fuzzy search input text |
| Text secondary | `#8d999a` (var(--text)) | Git file counts, inactive project names, modal field placeholders |
| Border | `#3e454a` (var(--border)) | Sidebar section dividers, modal border, input field borders |
| Badge: modified | `#b58900` (Solarized Yellow) | Modified file count badge |
| Badge: staged | `#859900` (Solarized Green) | Staged file count badge |
| Badge: untracked | `#6c7b83` (Solarized base01, muted) | Untracked file count badge |

Accent reserved for: active project left-border indicator (3px), Ctrl+P selected result background (10% opacity overlay), "Add Project" primary button background, git branch badge text color. Never used for all interactive elements indiscriminately.

## Component Inventory

### 1. Sidebar (rewrite of existing sidebar.js)

**Expanded state (200px):**
```
+---------------------------+
| EFXMUX              [+]  |  <- header row, "+" button right-aligned
|---------------------------|
| PROJECTS                  |  <- section label, 11px uppercase
|                           |
| | efx-mux          main  |  <- active: 3px left accent border, branch badge
|   my-api           dev   |  <- inactive: no left border
|   docs-site        main  |
|                           |
|---------------------------|
| GIT CHANGES          [R] |  <- section label + refresh icon button
|                           |
|  M 3  S 1  U 2           |  <- colored count badges inline
|                           |
|  src/main.js         M   |  <- file list, status letter right-aligned
|  src/lib.rs          M   |
|  new-file.txt        U   |
|  staged.js           S   |
+---------------------------+
```

- Active project row: `background: rgba(37, 138, 209, 0.08)`, `border-left: 3px solid var(--accent)`, `padding-left: 5px` (8px minus 3px border).
- Inactive project row: `padding: 6px 8px`, full 8px left padding.
- Project row height: auto (content-driven), minimum touch target 32px.
- Git branch badge: `font-size: 11px`, `color: var(--accent)`, right-aligned in project row.
- Git changes section: collapsible via click on section header. Default expanded.
- File list items: `font-size: 12px`, `padding: 4px 8px`, `cursor: pointer`. Hover: `background: var(--bg-raised)`.
- Status letter (M/S/U): right-aligned, colored per badge colors above.
- Section divider: `1px solid var(--border)`, `margin: 8px 0`.
- "+" button: 24x24px, `font-size: 16px`, `color: var(--text)`, hover: `color: var(--accent)`. Content: Unicode `+` (U+002B).
- Refresh button [R]: 20x20px, `font-size: 12px`, Unicode `\u21BB` (clockwise arrow). Same hover behavior as "+".

**Collapsed state (40px):**
- Show first letter of each project name as 24x24px centered icon.
- Active project icon: `color: var(--accent)`.
- Click any icon: expand sidebar AND highlight that project.
- Bottom of collapsed strip: "+" icon for add project.

### 2. Add Project Modal (new: project-modal.js)

**Layout:**
```
+-------- dimmed backdrop (rgba(0,0,0,0.5)) --------+
|                                                     |
|   +------------------------------------------+      |
|   | Add Project                         [X]  |      |
|   |------------------------------------------|      |
|   | Directory    [/path/to/project]    [...]  |      |
|   | Name         [project-name         ]      |      |
|   | Agent        [claude         v]           |      |
|   | GSD File     [optional path        ]      |      |
|   | Server Cmd   [optional command     ]      |      |
|   |                                          |      |
|   |              [Cancel]  [Add Project]      |      |
|   +------------------------------------------+      |
|                                                     |
+-----------------------------------------------------+
```

- Modal width: 480px, centered horizontally and vertically.
- Modal surface: `background: var(--bg-raised)`, `border: 1px solid var(--border)`, `border-radius: 4px`.
- Backdrop: `background: rgba(0, 0, 0, 0.5)`, covers full viewport, `z-index: 100`.
- Modal `z-index: 101`.
- Title: 16px, `color: var(--text-bright)`, `padding: 16px 24px 0`.
- Close [X] button: 24x24px, top-right corner of modal header, Unicode `\u2715`.
- Form fields: `padding: 0 24px`, `margin-top: 16px` between fields.
- Input fields: `width: 100%`, `height: 32px`, `padding: 0 8px`, `font-size: 14px`, `background: var(--bg)`, `border: 1px solid var(--border)`, `border-radius: 2px`, `color: var(--text-bright)`. Focus: `border-color: var(--accent)`, `outline: none`.
- Field labels: `font-size: 11px`, `text-transform: uppercase`, `letter-spacing: 0.08em`, `color: var(--text)`, `margin-bottom: 4px`.
- Directory field: has a browse button [...] that triggers native directory picker. Browse button: `width: 32px`, `background: var(--bg)`, `border: 1px solid var(--border)`, `border-left: none`, inline with input.
- Agent dropdown: native `<select>` styled to match inputs. Options: claude, opencode, bash.
- Button row: `padding: 16px 24px 24px`, `display: flex`, `justify-content: flex-end`, `gap: 8px`.
- Cancel button: `background: transparent`, `border: 1px solid var(--border)`, `color: var(--text)`, `padding: 6px 16px`, `border-radius: 2px`. Hover: `border-color: var(--text)`.
- Add Project button: `background: var(--accent)`, `color: #ffffff`, `border: none`, `padding: 6px 16px`, `border-radius: 2px`. Hover: `opacity: 0.9`. Disabled when directory field is empty.
- Animation: modal fades in over 150ms (`opacity: 0 -> 1`). No slide or scale animation.
- Escape key dismisses modal. Click on backdrop dismisses modal.

**First-run behavior (D-04):** When `projects` array is empty on startup, the modal opens automatically. The Cancel button is hidden in first-run mode (user must add at least one project). Close [X] button remains available.

### 3. Fuzzy Search Overlay (new: fuzzy-search.js)

**Layout:**
```
+-------- dimmed backdrop (rgba(0,0,0,0.3)) --------+
|                                                     |
|   +------------------------------------------+      |
|   | > search query here_                     |      |  <- top 20% of viewport
|   |------------------------------------------|      |
|   |   efx-mux                          main  |      |  <- selected (accent bg)
|   |   my-api                            dev  |      |
|   |   docs-site                        main  |      |
|   +------------------------------------------+      |
|                                                     |
+-----------------------------------------------------+
```

- Overlay width: 480px, centered horizontally, positioned at 20% from top of viewport.
- Surface: `background: var(--bg-raised)`, `border: 1px solid var(--border)`, `border-radius: 4px`, `box-shadow: 0 8px 32px rgba(0, 0, 0, 0.4)`.
- Backdrop: `background: rgba(0, 0, 0, 0.3)`, `z-index: 100`.
- Overlay `z-index: 101`.
- Input row: `padding: 12px 16px`, `font-size: 16px`, `color: var(--text-bright)`, `background: transparent`, `border: none`, `border-bottom: 1px solid var(--border)`. Prompt prefix `>` in `color: var(--accent)`, `margin-right: 8px`.
- Result list: max 8 visible items, scrollable if more. Each item `padding: 8px 16px`, `font-size: 14px`.
- Selected result: `background: rgba(37, 138, 209, 0.12)`. Arrow keys move selection.
- Result item shows: project name (left, `color: var(--text-bright)`) + git branch (right, `color: var(--accent)`, `font-size: 11px`).
- No results state: "No matching projects" in `color: var(--text)`, centered, `padding: 16px`.
- Animation: overlay fades in over 100ms.
- Escape key dismisses. Enter key switches to selected project and dismisses. Backdrop click dismisses.

### 4. Warning Toast (new: inline in main.js or toast utility)

- Position: bottom-center of viewport, 32px from bottom edge.
- Width: auto (content-driven), max 480px.
- Surface: `background: var(--bg-raised)`, `border: 1px solid var(--border)`, `border-radius: 4px`, `padding: 12px 16px`.
- Error variant: `border-left: 3px solid #dc322f`.
- Info variant: `border-left: 3px solid var(--accent)`.
- Text: `font-size: 13px`, `color: var(--text-bright)`.
- Auto-dismiss: 4 seconds. No manual dismiss button (keeps it simple).
- Animation: slide up 8px + fade in over 150ms. Fade out over 150ms before removal.
- `z-index: 200` (above modals).

## Interaction States

### Sidebar Project Row
| State | Visual |
|-------|--------|
| Default (inactive) | `color: var(--text)`, no left border |

| State | Visual |
|-------|--------|

| State | Visual |

| Focus | `border-color: var(--accent)` |

| State | Visual |
|-------|--------|

| Hover | `opacity: 0.9` |
| Disabled | `opacity: 0.4`, `cursor: not-allowed` (when directory field is empty) |

## Keyboard Contract

| Key | Context | Action |
|-----|---------|--------|
| Ctrl+P | App-level (captured before terminal) | Open fuzzy search overlay |
| Escape | Fuzzy search open | Close fuzzy search |
| Escape | Modal open | Close modal (except first-run where Cancel is hidden -- Escape still works via [X]) |
| Arrow Up/Down | Fuzzy search open | Navigate results |
| Enter | Fuzzy search open | Switch to selected project, close overlay |
| Enter | Modal focused, all required fields filled | Submit modal (add project) |

## Spacing Scale

**Inherited Project Scale** -- The project uses a compact, non-8-point spacing scale defined in `src/tokens.ts`, inherited from prior phases. These are pre-existing design tokens established before this phase and are intentional for dense developer tooling. This is a locked project constraint (see `tokens.ts` lines 58-69).

| Token | Value | Source | Usage in this phase |
|-------|-------|--------|---------------------|
| xs | 1px | tokens.ts | Not used |
| sm | 2px | tokens.ts | Minimal internal padding |
| md | 4px | tokens.ts | Icon-to-label gap in tabs, inline padding |
| lg | 6px | tokens.ts | Accordion header vertical padding |
| xl | 8px | tokens.ts | Tab bar horizontal padding (px-2), gap between tabs |
| 2xl | 10px | tokens.ts | Modal input padding |
| 3xl | 12px | tokens.ts | Accordion content padding, gap between action buttons |
| 4xl | 16px | tokens.ts | Tab horizontal padding, modal section gaps, form field gaps |
| 5xl | 20px | tokens.ts | Modal header padding |
| 6xl | 28px | tokens.ts | Not used |

Exceptions:
- Tab pill padding: `9px 16px` (vertical 9px matches existing TerminalTabBar pattern -- not a token value, inherited from prior phase implementation)

## Interaction Contracts

### Tab Drag-and-Drop (EDIT-05)

| Step | Behavior |
|------|----------|
| Drag start | Set `draggable` attribute on tab element. Store source index in `dataTransfer`. Set `effectAllowed: 'move'`. Apply 50% opacity to dragged tab. |
| Drag over | `preventDefault()` to allow drop. Set `dropEffect: 'move'`. Show 2px left border in accent color on hovered tab (insertion marker). |
| Drop | Read source index from `dataTransfer`. Reorder tabs array. Remove insertion marker. Persist new order. |
| Drag end | Restore full opacity. Remove all insertion markers. |
| Keyboard alternative | None required for Phase 17 (deferred). |

### Unsaved Changes Flow (EDIT-02, EDIT-03, EDIT-04)

| Trigger | Behavior |
|---------|----------|
| User edits buffer | CM6 `updateListener` fires on `docChanged`. Compare `doc.toString()` to saved content ref. If different, set tab `dirty: true` -- show yellow dot. If same, set `dirty: false` -- hide dot. |
| User presses Cmd+S | CM6 keymap intercepts `Mod-s`. Call `file-service.ts writeFile(filePath, doc.toString())`. On success: update saved content ref, set `dirty: false`, show success toast. On error: show error toast. |
| User clicks close (x) on dirty tab | Show ConfirmModal with three buttons: "Cancel" (dismiss modal, keep tab), "Discard" (close tab without saving), "Save File" (save then close). |
| User clicks close (x) on clean tab | Close tab immediately, no modal. |
| Cmd+S when no editor tab active | No-op (only intercept when active tab is editor type). |

### [+] Dropdown Menu (MAIN-01)

| Menu Item | Action | Icon |
|-----------|--------|------|
| Terminal (Zsh) | Create new terminal tab (calls existing `createNewTab()`) | `Terminal` from lucide-preact |
| Agent | Create new agent terminal tab (calls `createNewTab()` with agent binary) | `Bot` from lucide-preact |
| Git Changes | Create or focus git changes tab (singleton -- one instance only) | `FileDiff` from lucide-preact |

### File Open Flow (D-15, D-16)

| Step | Behavior |
|------|----------|
| User single-clicks file in tree | File tree dispatches `file-opened` custom event with `{ path, name }` |
| Event handler in main.tsx | Checks if editor tab for that path already exists (D-03 one-tab-per-file). If yes, focus existing tab. If no, create new EditorTab: invoke `read_file_content`, detect language, mount CM6. |
| Remove file viewer overlay | Delete `show-file-viewer` event handler and the overlay JSX in main-panel.tsx (D-16). |

### Tab Overflow

| Condition | Behavior |
|-----------|----------|
| Tabs fit in available width | Normal layout, no overflow indicators |
| Tabs exceed available width | Tab bar scrolls horizontally via `overflow-x: auto`. Each tab has `max-width: 200px` with `text-overflow: ellipsis` on the label. Native scrollbar is hidden via CSS (`scrollbar-width: none`, `-webkit-scrollbar { display: none }`). |
| Mouse wheel on tab bar | Horizontal scroll via `wheel` event with `deltaY` mapped to `scrollLeft` |

# Phase 6.1 Plan 03: Convert 5 UI Components to Preact TSX Summary

Converted tab-bar, main-panel, diff-viewer, gsd-viewer, and file-tree from Arrow.js html templates to Preact TSX function components with typed props, Tailwind utility classes, and @preact/signals state management.

## What Was Done

### Task 1: tab-bar.tsx and main-panel.tsx (ba0b8c5)

**tab-bar.tsx** - Simplest component, 26 lines. Arrow.js `html` template with `@click` and reactive class expression converted to Preact JSX with `onClick` and ternary class logic. Props typed with `Signal<string>` for activeTab to enable fine-grained signal subscription. Tailwind utility classes replace inline styles.

**main-panel.tsx** - Terminal area + file viewer overlay + server pane. Module-level signals replace Arrow.js `reactive()` object. `useEffect` registers `show-file-viewer` and `keydown` event listeners with proper cleanup. Conditional rendering uses `{fileViewerVisible.value && (...)}` instead of Arrow.js `style="display: ${() => ...}"`. Preserves `terminal-area` class name and `data-handle="main-h"` attribute required by terminal-manager.ts and drag-manager.ts.

### Task 2: diff-viewer.tsx, gsd-viewer.tsx, file-tree.tsx (5b18da8)

**diff-viewer.tsx** - `useRef<HTMLDivElement>` replaces `document.getElementById('diff-viewer-content')`. `useEffect` registers `open-diff` CustomEvent listener with cleanup. `escapeHtml()` and `renderDiffHtml()` preserved as pure typed helper functions (T-06.1-04 mitigation).

**gsd-viewer.tsx** - Imports `activeProjectName` and `projects` signals directly from state-manager.ts instead of receiving `activeProject` getter as parameter. `useEffect` manages three listeners (md-file-changed Tauri event, project-changed DOM event, initial load) with proper cleanup. `marked.parse` with custom checkbox renderer preserved. Checkbox write-back via `invoke('write_checkbox')` preserved with error revert (T-06.1-03 mitigation).

**file-tree.tsx** - Module-level signals for entries, selectedIndex, currentPath, loaded. Keyboard navigation via `onKeyDown` prop. Backspace guard preserved (only navigate up within project root). `file-opened` CustomEvent dispatch preserved.

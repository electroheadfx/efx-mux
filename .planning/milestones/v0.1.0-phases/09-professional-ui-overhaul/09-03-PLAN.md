   ```html
       <span class="text-success">+{addCount}</span>
       <span class="text-text mx-1">/</span>

     </span>
   </div>
   ```

3. Render diff lines in a container below the header:
   ```html
   <div class="border border-t-0 border-border rounded-b-md overflow-hidden mb-2">
   ```

   - Context (` `): `<div class="flex"><span class="w-12 text-right pr-2 text-[11px] text-text/50 select-none shrink-0">{lineNo}</span><div class="flex-1 pl-2 text-text">{escaped}</div></div>`

   ```tsx
   function formatSize(bytes: number): string {

     return `${(bytes / (1024 * 1024)).toFixed(1)}M`;
   ```

     ```javascript
     const terminals = []; // Array of { terminal, fitAddon }
     export function registerTerminal(terminal, fitAddon) { terminals.push({ terminal, fitAddon }); }
     export function unregisterTerminal(terminal) {
       const idx = terminals.findIndex(t => t.terminal === terminal);
       if (idx !== -1) terminals.splice(idx, 1);
     }
     ```

   - **`applyTheme(theme)`** function:
     - If `theme.chrome` exists, set CSS custom properties on `document.documentElement.style`:
       - `--bg` = `theme.chrome.bg`
       - `--bg-raised` = `theme.chrome.bgRaised`
       - `--border` = `theme.chrome.border`
       - `--text` = `theme.chrome.text`
       - `--text-bright` = `theme.chrome.textBright`
       - `--accent` = `theme.chrome.accent`
       - `--font` = `'${theme.chrome.font}', monospace` (if `theme.chrome.font` exists)
       - `--font-size` = `${theme.chrome.fontSize}px` (if `theme.chrome.fontSize` exists)
     - If `theme.terminal` exists, iterate over `terminals` array:
       - Set `term.terminal.options.theme = theme.terminal` (RESEARCH Pitfall 2: full replace, always send complete object)
       - If `theme.chrome?.font`, set `term.terminal.options.fontFamily = '${theme.chrome.font}', monospace'`
       - If `theme.chrome?.fontSize`, set `term.terminal.options.fontSize = theme.chrome.fontSize`
       - Call `term.fitAddon.fit()` after every theme application (RESEARCH Pitfall 4: font change requires fit)

   - **`getTerminalTheme()`** function that returns the last loaded `theme.terminal` object (cached from last applyTheme call). Used by terminal-manager.js for initial Terminal creation. Store in module-level `let currentTheme = null;`

   - **`toggleThemeMode()`** function (per D-14):
     ```javascript
     export function toggleThemeMode() {
       const root = document.documentElement;
       const current = root.getAttribute('data-theme') || 'dark';
       const next = current === 'dark' ? 'light' : 'dark';
       root.setAttribute('data-theme', next);
       localStorage.setItem('efxmux:theme-mode', next);
     }
     ```

   - **`initTheme()`** async function:
     - Restore dark/light mode from localStorage FIRST (before any paint):
       ```javascript
       const savedMode = localStorage.getItem('efxmux:theme-mode') || 'dark';
       document.documentElement.setAttribute('data-theme', savedMode);
       ```
     - Call `const theme = await invoke('load_theme')` via `window.__TAURI__.core.invoke`
     - Call `applyTheme(theme)`
     - Set up hot-reload listener: `await listen('theme-changed', (event) => { applyTheme(event.payload); })` via `window.__TAURI__.event.listen`
     - Return the loaded theme (so main.js can use terminal section for initial terminal creation)

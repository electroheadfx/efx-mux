## Milestone Summary

**Key Decisions:**
- CodeMirror 6 for editor tabs (lightweight, tree-shakeable)
- Unified tab bar across main and right panels with scope-aware state
- Cross-scope drag-and-drop for all tab types
- SubScopePane with always-mount bodies (display:none toggle) to preserve xterm/CM state
- Monotonic PTY counter for stable session names

**Issues Resolved:**
- File tree not updating when external editors modify files
- Open-in-editor button failures
- Tab persistence across restarts
- Cross-scope tab drag producing wrong content

**Technical Debt Incurred:**
- Audit tool bug: looks for SUMMARY.md not *-SUMMARY.md pattern
- 37 debug session files not formally closed
- Some UAT/verification checkboxes not marked complete
